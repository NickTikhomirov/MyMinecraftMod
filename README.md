# Фичи мода (Ru Wiki)

## Предметы с функционалом:
### Метла
```
  Не ломается
  Превращает траву в травяной путь (как лопата)
  Наносит дополнительный урон паукам
  Быстро ломает паутину
  При ударе по железному/снежному голему разламывает их на составляющие части
  Книжные полки, сундук края, стол зачарования
    а) ломает быстро
    б) ломает с шёлковым касанием
```
### Простой лазер
```
  Бесконечная зажигалка
```
### Похититель
```
  Shift+ПКМ по Деревенскому жителю телепортирует содержимое его инвентаря в ваш сундук края
  !Внимание, у предмета пока неправильно работают частицы - сервер не ляжет, но и глаз не порадуется
```
### Камушек
```
  Можно бросить - нанесёт 1 сердечко физическим уроном
  При промахе останется лежать на месте падения в виде предмета
```
### Магический камушек
```
  Можно бросить - нанесёт 1 сердечко магическим уроном
  При промахе возвращается вам в инвентарь
  За пределами обычного мира, ада и края наносит 1.5 сердечка
  Убивает с одного удара гастов (если у них не больше 12 сердечек)
    Добычу с них телепортирует вам в инвентарь (не распространяется на опыт и кристаллы заихрониума)
  Свинозомби наносит 4 сердечка магического урона "от себя" - т.е. не от игрока
  Урон по криперам - 2 сердечка "от скелета"
```
### Амулет зайхрониума (Xycronium) - если установлен ProjectX 2
```
  Имеет силу только в левой руке или слоте талисмана из Baubles
  Позволяет находить кристаллы зайхрониума в разных местах
    с любого босса падает по 16-20 кристаллов каждого типа
    с чешуйницы падает 1 кристалл случайного типа
    со скелета-иссушителя падает 1-3 чёрных кристалла (шанс: 20%)
    со стража падет 2-3 синих кристалла
    с древнего стража падает 
      10-11 синих кристаллов
      10-11 белых кристаллов
    с гаста падает
      1-2 белых кристаллов (шанс: 50%)
      2-3 красных кристаллов (шанс: 20%)
      1-2 черных кристаллов (шанс: 20%)
    с угольной руды падает чёрный кристалл (шанс: 1%)
    с руды редстоуна падает красный кристалл (шанс: 5%)
    с кварцевой руды падает белый кристалл (шанс: 10%)
    с алмазной руды падает белый кристалл
    с изумрудной руды падает зелёный кристалл
    с радужной руды падает по одному кристаллу каждого цвета
    (остальные источники ещё в разработке)
```

## Эффекты:
### Защита разума
```
  Снимает эффекты слепоты и тошноты, а также
  2 уровень: снимает усталость, медлительность и слабость
```
### Преобразование магии
```
  Весь магический урон считается огненным, а также
  2 уровень: уменьшает его на 1 сердечко
```
### Защита от иссушения
```
  Снимает эффект иссушения, а также
  1 уровень: уменьшает урон типа "иссушение" в 2 раза
  2 уровень: обнуляет урон типа "иссушение" и увеличивает свою длительность на 30 секунд при получении урона от взрыва
```
### Защита от края
```
  Снимает эффект левитации, при получении урона от наковальни пересчитывает свою длительность как
  "длительность = длительность*уровень",  а также
    При получении урона от чешуйницы края: отменяет урон и наносит ей(чешуйнице края) 50 урона
    При получении урона от Дракона Края: уменьшает урон по сердечку за каждый уровень
    При получении урона от странника края: уменьшает урон на 2^уровень, где 2 - одно сердечко
    При получении урона от кристалла края: отменяет урон
    При получении урона от шалкера: уменьшает урон по 1 сердечку за уровень и, если итоговый урон отрицательный,
       то "избыток" наносится уже самому шалкеру
  , а если цель эффекта является
    Странником края: каждые полсекунды с шансом 33% наносит ему 1+(уровень/2) сердечек 
        и накладывает слабость уровня (уровень-1) (т.е. при уровень=1 не накладывает)
    Чешуйницей: накладывает левитацию
    Чешуйницей края: наносит 250 сердечек
    Шалкером: каждые полсекунды с шансом 33% наносит 1+уровень сердечек
```
### Защита от ядов
```
  Чума не наносит урон (но может распространяться), а также
  Отравление, тошнота, голод, кислота
    1 уровень: снимает
    2 уровень: заменяет по правилу
        Голод -> Насыщение
        Отравление -> Регенерация
        Тошнота -> Подводное дыхание
        Кислота - просто отменяет урон
```
### Человечность
```
  Уменьшает урон от зомби (и всех производных от него) на (уровень+x) сердечек, где
  x=0.5 в аду
  х=1 вне ада
  
  Эффект снимается, если съесть исцелённую плоть
```
### Чума
```
  Каждую секунду наносит цели уровень+0.5 сердечка
  С некоторым шансом может распространиться на окружающих врагов (в области 6х6х4)
  Если на исходного носителя действует эффект человечности, то не может распространиться на игроков
  При распространениии также "берёт с собой" и эффект человечности
  Не наносит урон целям с защитой от ядов
  Снимается огнём
  У "дочерних" эффектов длительность всегда будет на 1 тик меньше исходного
```
### Кислота
```
  Мгновенный урон, по своему действию идентичный зелью мгновенного урона
```

## Зелья:
```
  Спешки (обычное/длинное)
    Спешка 2 (3мин/8мин)
  Защиты разума (обычное/сильное)
    Защита разума 1 (1мин/нет)
    Защита разума 2 (нет/8мин)
  Преобразования магии (обычное/сильное)
    Преобразование магии 1 (1.5мин/нет)
    Преобразование магии 2 (нет/5мин)
  Защиты от края (обычное/длинное/сильное/короткое)
    Защита от края 1 (3мин/8мин/нет/нет)
    Защита от края 2 (нет/нет/3мин/нет)
    Защита от края 3 (нет/нет/нет/1мин)
  Защиты от иссушения (обычное/длинное/сильное)
    Защита от иссушения 1 (3мин/8мин/нет)
    Защита от иссушения 2 (нет/нет/3мин)
  Человечности (обычное/длинное/сильное)
    Человечность 1 (3мин/8мин/нет)
    Человечность 2 (нет/нет/3мин)
  Кислота (обычная/сильная/сильная+)
    Кислота (1ур/2ур/3ур)
  
  Пожирания разума
    Слепота на 4 минуты
    Тошнота на 4 минуты
  Океанское зелье (обычное/длинное/боевое)
    Ночное зрение (2.5мин/7.5 мин/7мин)
    Подводное дыхание (2.5мин/7.5 мин/7мин)
    Защита разума 2 (нет/нет/7мин)
  Золотое яблоко (обычное, зачарованное)
    Все эффекты соответствующего яблока на то же время
    
  Зелье "...зелье?" (на самом деле не зелье)
    Кладёт в инвентарь 6-13 слёз гаста и накладывает <сюрприз>
```
